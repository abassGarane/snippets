## Snippets
## ***Uses***
- Golang
- Gohtml templates
- Mysql database

## Running software
- Start by creating a dockerized version of mysql
```terminal
cd snippets/
```
```bash
docker-compose up -d
```

- Run the program
```bash
go run cmd/api/!(*_test).go
```
- View project on ***`https/localhost:4000`***
